# Firebase 인증 구현 진행 상황

## 현재 구현된 기능
- 기본적인 이메일/비밀번호 회원가입 UI
- Firebase Authentication 연동
- Firestore 사용자 정보 저장
- 기본적인 로그인 UI

## 해결해야 할 문제
1. Firebase 설정 관련
   - reCAPTCHA 설정 문제 (CONFIGURATION_NOT_FOUND 오류)
   - SHA-1 인증서 지문 등록 필요
   - Firebase 초기화 설정 최적화 필요

2. 보안 관련
   - reCAPTCHA 인증 구현 필요
   - 비밀번호 정책 강화 필요
   - 이메일 인증 기능 추가 필요

3. UI/UX 개선사항
   - 로딩 상태 표시 개선
   - 에러 메시지 더 자세히 표시
   - 비밀번호 재확인 필드 추가
   - 비밀번호 보기/숨기기 기능

4. 추가 필요 기능
   - 비밀번호 재설정
   - 소셜 로그인 (Google, 카카오 등)
   - 자동 로그인
   - 로그아웃 기능
   - 회원 탈퇴 기능
